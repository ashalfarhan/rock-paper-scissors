# Rock Paper Scissors

A simple web-based game.

## Tech Stack

- [SvelteKit](https://kit.svelte.dev)

## Run Locally

Clone the project

```bash
git clone https://github.com/ashalfarhan/rock-paper-scissors.git
```

Go to the project directory

```bash
cd rock-paper-scissors
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
